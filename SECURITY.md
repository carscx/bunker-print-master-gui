# Security Policy

## Alcance

Este repositorio distribuye una aplicacion de escritorio para Windows.

## Buenas practicas para usuarios

1. Descargar binarios solo desde GitHub Releases oficial.
2. Verificar SHA256 antes de ejecutar instaladores.
3. Revisar el codigo fuente si necesitas auditoria adicional.

## Reporte de vulnerabilidades

Si detectas un problema de seguridad:

1. No publiques exploit detallado en un issue abierto inicialmente.
2. Contacta al mantenedor del proyecto y comparte reproduccion minima.
3. Se publicara fix y nota en CHANGELOG.

## Sobre riesgo

Publicar codigo fuente y checksums mejora la trazabilidad y reduce riesgo de binarios manipulados.
Aun asi, ningun software puede prometer riesgo cero absoluto.
